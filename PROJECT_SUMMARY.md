# 炸鸡店经营管理系统 - 项目交付说明

## 项目概述

**项目名称**: 炸鸡店经营管理系统 (Fried Chicken Store Management System)

**GitHub仓库**: https://github.com/RussellCooper-DJZ/fried-chicken-manager

**项目描述**: 基于Python和Tkinter开发的图形界面应用，帮助炸鸡店经营者进行利润计算、成本分析和经营决策。

## 核心功能

### 1. 净利润计算
- 基于客单价、单量、营业天数等参数计算月度净利润
- 详细的成本分解（直接成本、固定成本、变动成本）
- 关键指标展示（毛利率、净利率、盈亏平衡点）
- 智能预警（亏损提示和改进建议）

### 2. 城市级别预设
- **一线城市**: 高成本高客单价模型（房租30000元，客单价35元）
- **二线城市**: 中等成本标准模型（房租15000元，客单价25元）
- **三线城市**: 低成本经济模型（房租8000元，客单价18元）
- 一键加载预设参数，快速开始分析

### 3. 敏感性分析
- 分析客单价±20%、单量±20%共25种组合的利润影响
- 生成详细的分析表格
- 绘制敏感性热力图（直观展示最优经营区间）
- 帮助决策价格调整和营销投入

### 4. 数据可视化
- **成本结构饼图**: 展示营业额分配结构
- **收支对比柱状图**: 对比营业额、总成本和净利润
- **敏感性热力图**: 展示不同价格和单量组合下的利润分布

### 5. 历史记录
- 自动保存每次分析结果到JSON文件
- 记录时间戳、输入参数和计算结果
- 支持数据追溯和趋势分析

## 计算模型

### 核心公式
```
净利润 = (客单价 × 单量 × 营业天数) × (1 - 直接成本比例) - (固定成本 + 变动成本)
```

### 成本分类
1. **直接成本**: 食材、包装等，按营业额比例计算（35%-45%）
2. **固定成本**: 房租 + 人工 + 水电（月度固定支出）
3. **变动成本**: 按单量计算的包装、平台费等（2-5元/单）

### 关键指标
- **毛利率** = (营业额 - 直接成本) / 营业额 × 100%
- **净利率** = 净利润 / 营业额 × 100%
- **盈亏平衡点** = 固定成本 / (客单价 × (1 - 直接成本比例) - 变动成本/单)

## 使用场景

### 1. 新店开业评估
- 输入预计的客单价和单量
- 评估不同城市的盈利可能性
- 确定最低单量要求

### 2. 价格调整决策
- 使用敏感性分析评估降价促销的效果
- 找到价格和单量的最优平衡点
- 避免盲目降价导致利润下降

### 3. 成本优化
- 对比不同成本结构下的利润情况
- 识别成本优化的关键环节
- 制定成本控制策略

### 4. 经营监控
- 定期输入实际数据进行分析
- 对比预期和实际的差异
- 及时调整经营策略

## 安装和运行

### 环境要求
- Python 3.7 或更高版本
- tkinter（通常随Python自带）
- matplotlib

### 安装步骤
```bash
# 1. 克隆仓库
git clone https://github.com/RussellCooper-DJZ/fried-chicken-manager.git
cd fried-chicken-manager

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python chicken_manager.py
```

### 快速开始
1. 启动程序后，选择城市级别
2. 点击"加载预设"按钮
3. 根据实际情况调整参数
4. 点击"计算净利润"查看分析结果
