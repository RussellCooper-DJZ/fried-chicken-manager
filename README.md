# 炸鸡店经营管理系统

一个基于Python和Tkinter开发的炸鸡店经营分析工具，帮助经营者进行利润计算、成本分析和敏感性评估。

## 功能特性

### 核心功能
- **净利润计算**: 基于客单价、单量、成本等参数精确计算月度净利润
- **城市级别预设**: 内置一线/二线/三线城市的标准经营参数
- **敏感性分析**: 分析客单价和单量波动对利润的影响
- **盈亏平衡点**: 自动计算达到盈亏平衡所需的最低单量
- **数据可视化**: 提供成本结构饼图、收支对比柱状图和敏感性热力图
- **历史记录**: 自动保存每次分析结果，支持数据追溯

### 计算模型

**核心公式**:
```
净利润 = (客单价 × 单量 × 营业天数) × (1 - 直接成本比例) - (固定成本 + 变动成本)
```

**成本分类**:
- **直接成本**: 食材、包装等，按营业额比例计算（35%-45%）
- **固定成本**: 房租、人工、水电等月度固定支出
- **变动成本**: 按单量计算的包装、外卖平台费等

## 使用指南

### 快速开始

1. **选择城市级别**: 从下拉菜单选择一线/二线/三线城市
2. **加载预设参数**: 点击"加载预设"按钮自动填充该城市的标准参数
3. **调整参数**: 根据实际情况修改客单价、单量、成本等参数
4. **计算分析**: 
   - 点击"计算净利润"查看详细经营报告
   - 点击"敏感性分析"评估价格和单量变动影响

### 参数说明

| 参数 | 说明 | 参考范围 |
|------|------|----------|
| 客单价 | 每单平均消费金额 | 18-35元 |
| 日均单量 | 每天平均订单数 | 80-150单 |
| 营业天数 | 每月营业天数 | 26-30天 |
| 房租 | 月度店铺租金 | 3000-30000元 |
| 人工成本 | 月度员工工资总额 | 8000-40000元 |
| 水电费 | 月度水电支出 | 1500-8000元 |
| 直接成本比例 | 食材包装占营业额比例 | 35%-45% |
| 变动成本/单 | 每单包装、平台费等 | 2-5元 |

### 城市预设参数

#### 一线城市
- 房租: 30,000元/月
- 人工: 40,000元/月
- 水电: 8,000元/月
- 成本率: 45%
- 建议客单价: 35元
- 建议日单量: 150单

#### 二线城市
- 房租: 15,000元/月
- 人工: 24,000元/月
- 水电: 4,500元/月
- 成本率: 40%
- 建议客单价: 25元
- 建议日单量: 100单

#### 三线城市
- 房租: 8,000元/月
- 人工: 15,000元/月
- 水电: 2,000元/月
- 成本率: 35%
- 建议客单价: 18元
- 建议日单量: 80单

## 功能演示

### 1. 净利润计算
输入经营参数后，系统将生成详细的经营分析报告，包括：
- 月营业额预估
- 成本明细（直接成本、固定成本、变动成本）
- 利润指标（毛利润、净利润、毛利率、净利率）
- 盈亏平衡点分析
- 经营建议

### 2. 可视化分析
- **成本结构饼图**: 直观展示各项成本占比
- **收支对比柱状图**: 对比营业额、总成本和净利润
- **敏感性热力图**: 展示不同价格和单量组合下的利润情况

### 3. 敏感性分析
系统会自动计算价格和单量在±20%波动范围内的25种组合情况，帮助您：
- 评估价格调整的影响
- 确定营销投入的必要性
- 找到最优的价格-单量组合

## 数据存储

程序会自动将每次分析结果保存到 `store_history.json` 文件中，包含：
- 分析时间戳
- 输入参数
- 计算结果
- 城市级别

历史数据可用于：
- 对比不同时期的经营状况
- 追溯决策依据
- 生成长期趋势报告
